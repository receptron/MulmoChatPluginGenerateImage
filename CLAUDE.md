# CLAUDE.md

## Plugin Overview

Image generation plugin for MulmoChat. Generate images using AI models.

## Common Guidelines

For standard plugin development guidelines, see:
https://github.com/receptron/GUIChatPluginTemplate/blob/main/CLAUDE.md

## Plugin-Specific Notes

### Features
- AI-powered image generation
- Preview and view components
- Integration with image editing workflow

### Dependencies
- `gui-chat-protocol`: Core protocol for GUI Chat plugins
- `@mulmochat-plugin/ui-image`: Shared image UI components (peerDependency)

### Related Plugins
- `@mulmochat-plugin/ui-image`: Provides shared Preview/View components
- `@gui-chat-plugin/edit-image`: For editing generated images
